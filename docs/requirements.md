# HMART Requirements

## Project Name

HMART

## Description

HMART is an enterprise e-commerce platform for businesses and individual customers.

The platform allows customers to purchase products from multiple categories and supports admin management and delivery partner operations.

---

## User Roles

### Admin

Admin can:

* Manage products
* Manage categories
* Manage inventory
* Manage customers
* Manage delivery partners
* Manage orders
* View analytics
* Generate reports
* Manage coupons
* Manage notifications

### Buyer

Buyer can:

* Register/Login
* Browse products
* Search products
* Add products to cart
* Checkout
* Pay online
* Track orders
* Download invoices
* Manage profile
* Manage addresses
* View order history

### Delivery Partner

Delivery partner can:

* Login
* View assigned orders
* Accept deliveries
* Update delivery status
* Mark delivered
* Upload proof of delivery
